## Contribution Guidelines

### How to Contribute
1. **Fork** the repository
2. **Create** a feature branch
3. **Add** high-quality icons (PNG, transparent background)
4. **Test** ZIP integrity and format compliance
5. **Submit** a Pull Request with detailed description

### Icon Submission Standards
- **Format**: PNG with transparent background
- **Quality**: Minimum 64x64 pixels, preferably higher
- **Naming**: Descriptive, lowercase, hyphen-separated
- **Source**: Legitimate VMware/Broadcom assets only
- **License**: Compatible with repository license

### Code of Conduct
All contributors must adhere to our [Code of Conduct](CODE_OF_CONDUCT.md):
- Respectful communication
- Inclusive environment
- Professional behavior
- Constructive feedback
